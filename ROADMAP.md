- Clean public repository with safe templates
- setup.sh as generic template
- Docker + Swarm templates
- Whitelisted generic skills only

## Phase 2 — Core Features (in progress)
- GitHub full agent (auto issues + PRs from error_patterns)
- Optimized reasoning engine (currently in development)
- Self-hosted Matrix / Element bridge (full chat rooms)
- Nextcloud full integration (calendar, tasks, notes, private-dev folder)
- MQTT broker full control (ESP32, Zigbee, scenes without HA)
- Disaster recovery scripts (already in disaster-recovery/)

## Phase 3 — Advanced Capabilities (planned)

- iDRAC / Redfish / IPMI full control
- Bank API read-only alerts (budget + energy correlation)
- Self-hosted SearXNG proxy for web search (zero token usage)

## Phase 4 — Future Vision
- Multi-agent internal system
- PWA mobile app with voice
- Full autonomous homelab management
- Public SaaS version (with integrity checks)
