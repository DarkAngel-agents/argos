## REST API
```bash
curl -H "Authorization: Bearer <token>" http://11.11.11.201:8123/api/states
curl -H "Authorization: Bearer <token>" http://11.11.11.201:8123/api/states/<entity_id>
curl -X POST -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"entity_id": "<entity>"}' \
  http://11.11.11.201:8123/api/services/<domain>/<service>
```

## Entitati utile
```bash
# Verifica stare
curl -s -H "Authorization: Bearer <token>" \
```
